# AI文本润色助手（Coze实现）

## 📌 项目简介
本项目基于 Coze 平台开发，实现了一个中文文本优化助手，可以将用户输入的普通表达优化为更专业、清晰、自然的表达。

## 🚀 核心功能
- 输入任意中文句子
- 自动输出优化后的表达
- 保持原意不变
- 提升表达专业度和流畅度

## 🧠 技术原理
本项目基于以下AI应用结构实现：

- Prompt Engineering：通过精确设计提示词控制模型输出风格
- LLM（大语言模型）：使用Coze内置模型进行文本生成
- 输入输出映射：用户输入 → 模型处理 → 优化结果输出

## 🛠 技术栈
- Coze（AI Agent平台）
- Prompt Engineering

## 📷 项目演示

### 示例1
输入：我觉得这个功能还可以再改一下  
输出：我认为该功能仍有进一步优化的空间。

## 💡 项目收获
- 理解了AI应用的基本结构（Prompt + 输入输出）
- 掌握了基础的Prompt设计方法
- 熟悉了Coze平台的基本使用流程

- 接入知识库实现上下文优化
